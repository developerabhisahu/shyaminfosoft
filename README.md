# SahuSphere Technology - School Management System

This is a modern, responsive React application for a School Management System, built with **Vite** and **Tailwind CSS**.

## Features

*   **Modern UI/UX**: Designed with a "pro-level" aesthetic using Tailwind CSS.
*   **Responsive Design**: Fully responsive layout for Desktop, Tablet, and Mobile.
*   **Smooth Animations**: Integrated with GSAP and Lenis for smooth scrolling and interactions.
*   **SPA Routing**: Uses `react-router-dom` for seamless page navigation.

## Tech Stack

*   **Frontend**: React (Vite)
*   **Styling**: Tailwind CSS v4
*   **Animations**: GSAP, Lenis
*   **Icons**: React Icons

## Getting Started

1.  **Clone the repository:**
    ```bash
    git clone <repository-url>
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Run the development server:**
    ```bash
    npm run dev
    ```

## Deployment

### Render
This project is configured for deployment on [Render](https://render.com).
It includes a `render.yaml` file to automate the build and routing configuration.

*   **Build Command**: `npm install; npm run build`
*   **Publish Directory**: `dist`
